## Sumscope Web 初始化脚手架

---

### Install 
```bash
$ npm i ss-web-cli -g
```

### Usage

Generate demo 

```bash
$ mkdir demo && cd demo
$ ss-web-cli
```

Start development server
```bash
$ npm start
```

Build production
```bash
$ npm run deploy
```

Build qa
```bash
$ npm run deploy:qa
```

Build dev
```bash
$ npm run deploy:dev
```
